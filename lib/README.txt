Letakkan file MySQL Connector/J di folder ini.

Contoh nama file:
- mysql-connector-j-8.0.33.jar
- mysql-connector-java-8.0.33.jar

Setelah file JAR diletakkan di folder `lib`, buka ulang VS Code atau jalankan `Java: Clean Java Language Server Workspace`,
lalu klik `Start Debugging` pada konfigurasi `MainApp`.
